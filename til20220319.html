3.19.1 일반선형델회귀모델

일반선형회귀모델

$ f(_y) = _0 + _1 x_1 + _2 x_2 + … + _n x_n $

로지스틱 선형회귀모델

$ ln( ) = _0 + _1 x_1 + _2 x_2 + … + _m x_m $

포아송 선형회귀모델

$ ln() = _0 + _1 x_1 + _2 x_2 + … + _m x_m $
